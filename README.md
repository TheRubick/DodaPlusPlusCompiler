# DodaPlusPlusCompiler
## To run the files

1. first compile the files using compile.sh
```
chmod +x compile.sh
./compile.sh
```
2. run the GUI
```
python3 GUI.py
```

## For cleaning the directory
```
rm y.tab.* && rm a.out && rm lex.yy.c
```

## check the requirements
```
sudo apt-get install python3-tk
```
